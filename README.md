This project is a basic Spring Boot application that provides CRUD operations for managing products. It uses a MySQL database for data storage and supports integration with Elasticsearch.

## Technologies Used

- **Java 17**
- **Spring Boot**
- **Spring Data JPA**
- **Hibernate**
- **MySQL**
- **Docker**
- **Gradle**

## Prerequisites

- Java 17 or higher
- Docker
- MySQL
- Gradle
